Quick Start

Include the library in a browser page and create an instance.

<script src="stringReverse.min.js"></script>
<script>
  const s = new StringUtils();
  console.log(s.reverseString('Hello'));
</script>

StringUtils (from stringReverse.js)

reverseString(str)
Reverses characters in a string. Throws if input is not a string.
const s = new StringUtils();
s.reverseString('abc') // 'cba'
countString(str)
Returns character length of the string.
new StringUtils().countString('Hello') // 5
capitalizeString(str)
Uppercases the first character (no-op for empty string).
new StringUtils().capitalizeString('hello') // 'Hello'

CommonUtils (from commonUtils.js)

The demo at demo.html shows live examples; below are concise usages.

String utilities

truncateString(str, length)
Truncates and adds '...'.
utils.truncateString('Hello World', 5) // 'Hello...'
slugify(str)
Lowercases, removes invalid chars and replaces spaces with '-'.
utils.slugify('Hi There!') // 'hi-there'

Array utilities

shuffleArray(arr)
Returns a shuffled copy.
utils.shuffleArray([1,2,3]) // e.g. [2,1,3]
uniqueArray(arr)
Returns deduplicated array.
utils.uniqueArray([1,1,2]) // [1,2]
chunkArray(arr, size)
Splits array into chunks.
utils.chunkArray([1,2,3,4], 2) // [[1,2],[3,4]]

Date & Time

formatDate(date, format)
Formats a Date with tokens: YYYY, MM, DD, HH, mm, ss.
utils.formatDate(new Date('2023-01-01'), 'YYYY-MM-DD') // '2023-01-01'
timeAgo(date)
Returns human readable elapsed time (e.g. '2 days ago').

DOM & Animation

toggleClass(el, className)
Toggles CSS class on an element.
fadeIn(el, duration)
Fades element in (demo in demo.html).

Cookies

setCookie(name, value, days) 
utils.setCookie('token', 'abc', 7)
getCookie(name) 
utils.getCookie('token') // 'abc' or null

HTTP (AJAX)

httpGet(url) 
const data = await utils.httpGet('https://jsonplaceholder.typicode.com/posts/1')
httpPost(url, data) 
const res = await utils.httpPost('/api', {a:1})

Form Validation

See form.html for a live example. Use `validateForm(formId, rules)` to run validation rules which return `{ isValid, errors }`.
// Example rules
const rules = { username: { required: true, minLength: 3 }, email: { required: true, email: true } };
const result = utils.validateForm('myForm', rules);
if (!result.isValid) console.log(result.errors);
